## 1. エグゼクティブサマリー

### 1.1 ユーザージャーニーマッピングとは

ユーザーがサービスを利用する際の一連の体験を可視化したものです。

### 1.2 Actoryの主要ジャーニー

1. **初回利用**: サインアップ → オンボーディング → 初回録音
2. **日常利用**: 録音 → 議事録確認 → タスク管理
3. **AI活用**: チャット → ナレッジ検索 → コンテンツ生成

---

## 2. 主要ユーザージャーニー

### 2.1 初回利用フロー

```
1. ランディングページ訪問
   ↓
2. サインアップ（Google認証）
   ↓
3. オンボーディング
   - 事業数入力
   - 事業名入力
   - 部門数入力
   ↓
4. ダッシュボード表示
   ↓
5. 初回録音開始
```

### 2.2 日常利用フロー

```
1. ダッシュボードログイン
   ↓
2. プロジェクト選択
   ↓
3. 録音開始
   ↓
4. 録音停止・アップロード
   ↓
5. 文字起こし完了通知
   ↓
6. 議事録確認
   ↓
7. タスク承認・編集
```
