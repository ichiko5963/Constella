# Constella ブランドコンセプト

## 名称
**Constella** (コンステラ)

## サブコピー
**"Connect context, like stars."**

---

## 語源と意味

### Constella = Constellation（星座）の派生
- **con-**（共に、集まって）
- **stella**（星）

→ 「星が共に在る状態」
→ 「点が集まり、関係性を持ったときに意味が生まれるもの」

---

## 3層の意味

### ① 点ではなく「関係性」
- 星単体には意味がない
- 配置・つながりによって初めて意味が立ち上がる
- メモや情報を「孤立した点」で扱わない

### ② 人が意味を見出す構造
- 星座は自然物ではなく、人が意味を見出したもの
- 文脈・解釈・認知が主役
- AIやプロダクトが「意味を与える」のではなく、意味が浮かび上がるのを支援する

### ③ 全体像を俯瞰できる
- 星座は「全体を見て初めて分かる」
- 時間・距離・流れをまたいで把握できる
- 思考・記憶・学習・人生ログと相性が良い

---

## プロダクト定義

> **Constella** =
> 「散らばった思考や記憶が、つながることで意味として見える場所」

| 用語 | 意味 | アイコン |
|------|------|----------|
| **Stella（ステラ）** | 個々のコンテキスト（情報の点） | ★ 星 |

## デザイン原則

### カラーパレット
- **Primary**: 白 (#FFFFFF)
- **Secondary**: 黒 (#000000)
- **Accent**: 淡いグレー (#F5F5F5, #E0E0E0)
- **Highlight**: 星の輝き（微かなゴールド/シルバー）

### デザインキーワード
- Minimal
- Elegant
- Starry
- Spacious (余白を活かす)
- Clean

### UI要素
- 星をモチーフにしたアイコン
- 星座を表す繊細な線
- 広がりを感じる余白
- 白を基調としたクリーンな背景

## スプラッシュアニメーション仕様

### コンセプト
「無数の星（Stella）が集まり、つながり、Constellaロゴを形成する」

### アニメーションフロー
1. **Phase 1: 星の出現** (0-0.5s)
   - 画面全体に小さな星がランダムに出現
   - フェードインしながら現れる

2. **Phase 2: 星の収束** (0.5-1.5s)
   - 星が中央に向かって移動
   - パーティクルエフェクトで軌跡を残す

3. **Phase 3: 星座形成** (1.5-2.5s)
   - 星同士が線で繋がる
   - Constellaロゴの形が浮かび上がる

4. **Phase 4: ロゴ完成** (2.5-3s)
   - ロゴが完全に形成
   - 微かなグロウエフェクト
   - フェードアウトしてメイン画面へ

### 技術実装
- Framer Motion + Canvas API
- WebGL（パフォーマンス最適化）
- CSS Animations（軽量版フォールバック）

## 録音UI/UX コンセプト

### テーマ
「星空の下での録音」

### 要素
- 録音中は星が瞬くアニメーション
- 音声レベルに応じて星が明滅
- 録音完了時に星が集まって文字起こしへ変換
